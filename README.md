# الآلة الحاسبة — Arabic Calculator

آلة حاسبة عربية بنمط iOS، تعمل بالكامل داخل ملف `index.html` واحد (HTML + CSS + JavaScript، بدون أي مكتبات خارجية).

## المميزات

- **عرض المعادلة**: تُظهر العملية كاملة أثناء الكتابة ثم النتيجة عند `=`، مع ترتيب الأولويات الصحيح.
- **وضع أساسي وعلمي**: مثلثات (جا/جتا/ظا) وعكسها عبر «ثانوي»، قوى وجذور، لوغاريتمات، مضروب، `π`/`e`، ذاكرة، عشوائي، وتبديل درجة/راديان.
- **سجل محفوظ**: يُحفظ في المتصفح (localStorage) ومجمّع حسب اليوم/أمس.
- **قائمة الأوضاع**: أساسية، علمية، الملاحظات الحسابية، تحويل الوحدات، أرقام عربية.
- **متجاوب بالكامل**: يتوافق مع الهاتف والآيباد والحاسوب، في الوضعين العمودي والأفقي.

## التشغيل

افتح `index.html` مباشرة في المتصفح، أو زر النسخة المنشورة عبر GitHub Pages.

A single-file, dependency-free iOS-style Arabic calculator with basic & scientific modes, saved history, unit conversion, and a fully responsive layout (portrait & landscape).
